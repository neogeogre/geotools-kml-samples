# geotools-kml-samples

Demo of kml manipulation with geotools written in Kotlin

## Java setup

Install `sdkman`
To autoload the `.sdkmanrc`:
in `$HOME/.sdkman/etc/config` set `sdkman_auto_env=true`


## How to build

```shell
./gradlew clean build 
```

```shell
./gradlew clean build -xtest 
```
